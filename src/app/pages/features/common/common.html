Common

Common interfaces and classes used by all components of this library.

Image

The main object that represents an Image with both modal, thumbnail, description and so on. The first thing to do is creating an array of Image instances (using the new keyword). Every element represents a single image.


id
Type: number

Mandatory UNIQUE id to identify the image.
modal
Type: ModalImage

Mandatory interface to configure the modal image.
plain
Type: PlainImageoptional

Optional interface to configure the thumbnail (or 'plain image').

DotsConfig

Interface to configure optional dots below the modal image.


visible
Type: boolean

If true, dots are visible, false otherwise. true by default.

Size

Interface to customize images/icon sizes.


width
Type: string

It can be a valid value for the css 'width' property.

For instance, you can use either '100px', 'auto', '20%'.

Default values change depending by the related component.
height
Type: string

It can be a valid value for the css 'height' property.

For instance, you can use either '100px', 'auto', '20%'.

Default values change depending by the related component.

Action

Enum that represents the source of an action. It can be one of these enum values:


  • NORMAL: the default value
  • CLICK: mouse click action
  • KEYBOARD: keyboard action (ctrl+s, cmd+s, esc, →, ← and so on)
  • SWIPE: swipe action
  • LOAD: data/images loaded by the library
  • AUTOPLAY: navigation caused by autoPlay
