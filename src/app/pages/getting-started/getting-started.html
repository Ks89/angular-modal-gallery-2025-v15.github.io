Supported Browsers

Starting from version 9.0.0 of this library, Internet Explorer 11 and legacy Microsoft Edge (pre-chromium) are not supported!


Installation

Based on the table above, install the required dependencies.

For the latest version of @ks89/angular-modal-gallery run

  • npm i --save @ks89/angular-modal-gallery@latest
  • npm i --save @angular/cdk

Please note that if you need angular-modal-gallery versions 5 or 3, you must use the older npm package called angular-modal-gallery without @ks89 scope.

All default icons don't require font-awesome, so starting from angular-modal-gallery 5.0.0 font-awesome is optional. However, if you want to use custom button icons, you are free to use either your own css or font-awesome (versions 4 or 5). Font Awesome 5 is officially supported from angular-modal-gallery 5.3.0 or greater.


Update your project

Once finished to install dependencies, you need to follow some simple steps.

I consider only angular-cli projects. If you want to use @angular/universal, please check official examples.


Step 1: Import dependencies

If you are working on a modern project with standalone components you can import GalleryModule with importProvidersFrom

Pass appConfig to bootstrapApplication

        bootstrapApplication(AppComponent, appConfig)
          .catch((err) => console.error(err));

Then use importProvidersFrom to load GalleryModule in this way

        import { GalleryModule } from '@ks89/angular-modal-gallery';

        export const appConfig: ApplicationConfig = {
          providers: [
            provideBrowserGlobalErrorListeners(),
            provideZoneChangeDetection({ eventCoalescing: true }),
            provideRouter(routes),
            importProvidersFrom(
              // import GalleryModule. Install @ks89/angular-modal-gallery first
              GalleryModule
            )
          ]
        };

Otherwise, if you are still working on a module based project, you have to list GalleryModule in your root module and any additional application modules that make use of the components of this library.

        import {GalleryModule} from '@ks89/angular-modal-gallery';
        @NgModule({
          imports: [
            GalleryModule,
            ...],
          declarations: [ AppComponent, ... ],
          providers: [ ... ],
          bootstrap: [ AppComponent ]
        })
        export class AppModule {}

Step 2: Update global CSS

Update your global styles (CSS/SCSS/LESS) file adding this:

        @use "~@angular/cdk/overlay-prebuilt.css" as *;
        .ks-modal-gallery-backdrop {
                  background: #000 !important;;
                  opacity: 0.85 !important;;
        }
        .ks-modal-gallery-panel {
                  z-index: 90000 !important;
        }

Please not that @import has been deprecated, so you have to use the new syntax @use ... as *. For more info check the official page here


Step 3: (optional) update tsconfig file

Add @types to tsconfig.json (in case on angular-cli project it's called tsconfig.json) as follow:

        {
          ...
          "compilerOptions": {
            ...
            "typeRoots": ["node_modules/@types"]
          }
        }
