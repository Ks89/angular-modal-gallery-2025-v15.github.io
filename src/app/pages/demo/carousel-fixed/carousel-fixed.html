Carousel with fixed width

Carousel with fixed width 766px, height 100px and 5 previews.

Also you can open modal gallery clicking on the current carousel image.

CarouselLibConfig


Input Value
carouselConfig
Value: { maxWidth: '766px', maxHeight: '400px', showArrows: true, objectFit: 'cover', keyboardEnable: true, modalGalleryEnable: true }
Fixed width 766px, all other properties have default values
carouselPreviewsConfig
Value: { number: 5, width: 'auto', maxHeight: '100px' }
It will show 5 clickable previews (height 100px) with arrows
