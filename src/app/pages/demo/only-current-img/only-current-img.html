ModalLibConfig object to config the library. See below.

ModalLibConfig


Input Value
buttonsConfig
Value: {visible: false, strategy: 1}
It will hide all buttons
dotsConfig
Value: {visible: false}
It will hide dots below the current modal image
previewConfig
Value: {visible: false}
It will hide previews below the current modal image
slideConfig
Value: {visible: false, sidePreviews: {show: false}}
Hide side previews and let infinite sliding to the default value (false)
