Download Simple example

Modal gallery example with downloadable images ONLY with ctrl+s/cmd+s.


Attention: for security reasons Google Chrome and other browsers decided to block cross-origin downloads. So to use the download feature of this example, your images must be in the same server of your Angular application, otherwise the browser will navigate to the image itself, without to download it automatically.
