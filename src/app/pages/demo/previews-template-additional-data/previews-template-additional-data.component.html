Previews custom template with additional data

Modal gallery example with custom previews, using an Angular template and custom image-related data.

This example demonstrates how to use custom image-specific data (here, the date of the picture) to render the previews.

previewsTemplate
Value: TemplateRef<HTMLElement>
A reference to an Angular template.
This template can take 2 template variables:
  • let-preview: gives access to the Image associated with the preview
  • let-defaultTemplate: gives access to the default template used to render the preview; helpful if the need is to wrap the preview with additional information. Insert the default template with: <ng-container *ngTemplateOutlet="defaultTemplate"></ng-container>.
