previewConfig
Type: PreviewConfigoptional

Optional object used to configure PreviewConfig.

Interface to configure previews, for instance to choose either the number, visibility, arrows, sizes.

Default value:

{ visible: true, number: 4, arrows: true, clickable: true, width: 100 / 4 + '%', maxHeight: '200px', breakpoints: { xSmall: 100, small: 100, medium: 150, large: 200, xLarge: 200 } }
dotsConfig
Type: DotsConfigoptional

Optional object used to configure DotsConfig.

Default value:

{ visible: true }
slideConfig
Type: SlideConfigoptional

Optional object used to configure SlideConfig to manage sliding features of modal-gallery and ....

Interface to enable infinite-sliding (you won't be able to navigate before both the first and after the last images) and to configure both side-previews and autoPlay. By default, infinite sliding is disabled, side-previews are visible with width: 100px and height: auto and autoPlay is disabled.

Default value:

{ infinite: false, playConfig: { autoPlay: true, interval: 5000, pauseOnHover: true }, sidePreviews: { show: true, size: { width: '100px', height: 'auto' } } }
accessibilityConfig
Type: AccessibilityConfigoptional

Optional object used to configure AccessibilityConfig.

Interface to customize accessibility features, like alt, titles, aria-labels and so on. By default all texts are written in english. This default config object is exported as KS_DEFAULT_ACCESSIBILITY_CONFIG.

Default value:

It's an object with all default english accessibility values. Check here for more information.
enableCloseOutside
Type: booleanoptional

Optional property to close modal-gallery clicking on the semi-transparent background.

If true, you can click on the semi-transparent background of modal images to close it. true by default.

Default value:

true
keyboardConfig
Type: KeyboardConfigoptional

Optional object used to configure KeyboardConfig.

Interface to configure keyboard's buttons. All properties are optional and with default values. Check KeyboardConfig for more information.

Default value:

esc, →, ←
currentImageConfig
Type: CurrentImageConfigoptional

Optional object used to configure CurrentImageConfig.

Interface to configure current image in modal-gallery. For instance you can either disable navigation when you click on the current image (enabled by default), or change descriptions, loading spinners or enable download features.

Default value:

{ invertSwipe: false, navigateOnClick: true, downloadable: false, loadingConfig: {enable: true, type: LoadingType.STANDARD}, description: { strategy: DescriptionStrategy.ALWAYS_VISIBLE, imageText: 'Image ', numberSeparator: '/', beforeTextDescription: ' - ', style: { bgColor: 'rgba(0,0,0,.5)', textColor: 'white', marginTop: '0px', marginBottom: '0px', marginLeft: '0px', marginRight: '0px' } } }
buttonsConfig
Type: ButtonsConfigoptional

Optional object used to configure ButtonsConfig.

Interface to choose which buttons to show and handle their behaviour.

Default value:

{ visible: true, strategy: ButtonsStrategy.DEFAULT }
plainGalleryConfig
Type: PlainGalleryConfigoptional

Optional object used to configure PlainGalleryConfig.

Interface to configure thumbnails, for instance to change the layout. Furthermore, it's useful to write full custom layouts (for more info see official demos).

Default value:

{ strategy: PlainGalleryStrategy.ROW, layout: this.defaultLayout, advanced: { aTags: false, additionalBackground: '50% 50%/cover' } }
carouselConfig
Type: CarouselConfigoptional

Optional object used to configure the carousel, like to change sizes, hide some features and so on.

For more information, please see the CarouselConfig's section

Default value:

{ maxWidth: '100%', maxHeight: '400px, showArrows: true, objectFit: 'cover', keyboardEnable: true, modalGalleryEnable: false }
carouselImageConfig
Type: CarouselImageConfigoptional

Optional object to change current image of the carousel, like description or to invert the swipe direction.

For more information, please see the CarouselImageConfig's section

Default value:

{ invertSwipe: false, description: { strategy: DescriptionStrategy.ALWAYS_HIDDEN, imageText: 'Image ', numberSeparator: '/', beforeTextDescription: ' - ', style: { bgColor: 'rgba(0,0,0,.5)', textColor: 'white', marginTop: '0px', marginBottom: '0px', marginLeft: '0px', marginRight: '0px' } } }
carouselPreviewsConfig
Type: CarouselPreviewConfigoptional

Optional object used to configure previews, for instance to choose either the number, visibility, arrows, sizes. If you want to know how to use this object, please check PreviewsComponent.

Default value:

{ visible: true, number: 4, arrows: true, clickable: true, width: 100 / 4 + '%', maxHeight: '200px', breakpoints: { xSmall: 100, small: 100, medium: 150, large: 200, xLarge: 200 }}
carouselPlayConfig
Type: PlayConfigoptional

Optional object used to change the behaviour of auto-navigation, like autoPlay and interval in milliseconds.

For more information, please see the PlayConfig's section

Default value:

{ autoPlay: true, interval: 5000, pauseOnHover: true }
carouselDotsConfig
Type: DotsConfigoptional

Optional object used to configure dots.

For more information, please see the DotsConfig's section

Default value:

{ visible: true }
carouselSlideInfinite
Type: booleanoptional

Optional property to configure infinite sliding of the carousel.

If true, carousel won't stop at the last image, but it will restart from the first image. true by default.

Default value:

true
